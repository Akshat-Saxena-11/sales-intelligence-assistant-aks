# Welcome to Akshat's Sales Intelligence Assistant!

## Project info

**URL**: https://sales-intelligence-assistant-aks.vercel.app/

## How can I use this web app?

Simply click the Vercel URL above and start asking questions based on the MXV AI Hackathon's sales policies or sales data.

## What technologies are used for this project?

The web app has been built using Lovable, with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

The web app is hosted on Vercel. The back-end of the chat interface consists of an n8n workflow which uses Retrieval Augmented Generation.
